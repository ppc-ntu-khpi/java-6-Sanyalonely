# Аналіз матриць: Пошук стовпця з максимальною сумою

## Опис завдання
Необхідно знайти стовпець матриці розміром `N x M` з **максимальною сумою елементів**.

## Реалізація та Оптимізація
Проєкт складається з двох класів, що знаходяться у теці `src`:
1. `MatrixProcessor.java` — універсальний клас, що містить виключно логіку проекту. Не містить інтерфейсу користувача та методів введення/виведення.
2. `Main.java` — тестовий клас для перевірки працездатності алгоритму на різних наборах даних.

### Ключові особливості розв'язку:
* **Без класичних циклів:** Обробку масивів та пошук максимуму реалізовано за допомогою **Java Stream API** (`IntStream`, `Arrays.stream`).
* **Оптимізація типів даних:** Сума елементів стовпця може легко перевищити ліміти стандартного типу. Тому для підрахунку сум використано тип `long`, щоб уникнути переповнення.
* **Документування:** Логіка добре задокументована за допомогою Javadoc-коментарів, які пояснюють суть методів.
